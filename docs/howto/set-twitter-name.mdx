# Set a Twitter name

Twitter does not yet integrate ENS names, or verify them, so setting them as your Twitter name is simpler than it first appears.
All you need to do is to edit your name and enter it in!

### Edit your Twitter profile
Go to <Link to="https://www.twitter.com"><b>Twitter</b></Link> and click on your `Profile`

<BrowserWindow url="https://twitter.com" bg="#000000" img="/img/howto/set-twitter-name-1.png"/>

Click the `Edit` button in the upper right of your profile page to edit it.
<BrowserWindow url="https://twitter.com" bg="#000000" img="/img/howto/set-twitter-name-2.png"/>

Type in your ENS name in the `Name` field
<BrowserWindow url="https://twitter.com" bg="#242d34" img="/img/howto/set-twitter-name-3.png"/>

Press `Save` to save it.
<BrowserWindow url="https://twitter.com" bg="#242d34" img="/img/howto/set-twitter-name-4.png"/>

That's it! You're done and should now be sporting your ENS name on Twitter!
<BrowserWindow url="https://twitter.com" bg="#000000" img="/img/howto/set-twitter-name-5.png"/>
