# Add a Custom Record
To add a custom record to your ENS name, open the nameâ€™s profile editor by clicking `Edit profile`

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-1-new1.png"/>

Next, click on the `+ Add more to profile` button.

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-2-new1.png"/>

On the "Add profile fields" window, click `other`. Then select the `Custom`, and click the `Add` button to create your custom record.

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-3-new1.png"/>

Type the name of the custom key into the first input box under `Custom`. Next to it, input the value you would like to save for the custom record and click `Save`.

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-4-new1.png"/>

On the next window click `Open Wallet` to confirm and sign the transaction. Your custom record will then be added to your ENS name!

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-5-new1.png"/>

## Add a Custom Service Key

Some types of ENS records use a key name formatted in "reverse dot notation". These are known as "Service Keys" as specified in <Link to="https://docs.ens.domains/ens-improvement-proposals/ensip-5-text-records#service-keys"><b>ENSIP-5</b></Link>. For example, you want to save your YouTube channel name into your ENS records. To create a custom record for the YouTube account service, simply name the key `com.youtube` and set its value to your YouTube channel name.

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-6-new1.png"/>

On the next window click `Open Wallet` to confirm and save the custom Service Key and its value to your ENS records. Now a record for YouTube.com will exist for your ENS name! There is no limit to the number or types of records you can create for your ENS name.

<BrowserWindow url="https://app.ens.domains" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-7-new1.png"/>
